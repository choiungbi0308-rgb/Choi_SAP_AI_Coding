## 1. Functional Specification

### 1.1 비즈니스 목적

ZMMR90810은 플랜트 / 저장위치 / 자재 단위로 **일별 재고 입출고 현황을 집계하는 MM 모니터링 리포트**이다. 지정 날짜 범위의 자재문서(MSEG)에서 모든 이동을 수집하고, 커스텀 이동유형 카테고리 마스터(ZMMT90800)에 따라 입고/출고/조정/전용 등으로 분류한다.

---

### 1.2 비즈니스 프로세스

```
플랜트 + 날짜 지정
       │
       ▼
MSEG 자재문서 조회 (ADBC, MKPF join)
       │
       ▼
ZMMT90800 카테고리 마스터 매핑 (1/2/3차 분류)
       │
       ├─ 이동유형별 입고(SHKZG='S') / 출고(SHKZG='H') 수량 집계
       │
       ▼
ZMMT80011 일별 재고 스냅샷 조회
       ├─ D-Day 재고 (LABST + INSME + SPEME)
       └─ D-1  재고 (LABST + INSME + SPEME)
       │
       ▼
ALV Grid 출력 (자재번호 더블클릭 → 개별 자재문서 드릴다운)
```
